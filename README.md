# Proyecto Solidario: Construyendo con Astro para KMAKUS
## Introducción
Este proyecto nace de una idea sencilla pero poderosa:
usar la tecnología para crear un impacto real.

Vamos a construir una página web desde cero utilizando Astro, un framework moderno, rápido y orientado al contenido, mientras retransmitimos todo el proceso en directo.
Pero este proyecto no solo va de código: va de ayudar.

## ¿Por qué KMAKUS?
**KMAKUS** es una organización que empezó como protectora y que hoy funciona como un centro de Adopciones.
Su misión es clara: ofrecer a cada mascota todos los servicios necesarios para encontrar un hogar definitivo y acompañar a las familias durante el proceso de adopción.

Creemos que su labor es imprescindible y queremos aportar desde aquello que mejor sabemos hacer: crear tecnología y compartir conocimiento.

Con este proyecto buscamos:
* Dar visibilidad al trabajo diario de KMAKUS.
* Atraer a más adoptantes y colaboradores.
* Promover la adopción responsable.
* Demostrar cómo el desarrollo web y la solidaridad pueden ir de la mano.

## Los Directos: Aprende Astro desde cero
Durante el proyecto realizaremos una serie de directos donde podrás ver:

* Cómo se construye una web real usando Astro.
* Prácticas modernas de desarrollo front-end.
* Integración de componentes, rutas, layouts y optimización.
* Herramientas, librerías y buenas prácticas.
* Cómo se transforma una idea solidaria en un proyecto funcional.

Los directos están pensados tanto para personas que empiezan en el desarrollo web como para quienes ya tienen experiencia y quieren aprender Astro de forma aplicada.

## ¿Por qué en directo?
Porque queremos:

* Enseñar de forma transparente.
* Responder preguntas en tiempo real.
* Compartir conocimiento como comunidad.
* Mostrar el proceso completo sin filtros: desde los aciertos hasta los errores.

## Cómo puedes ayudar
Hay muchas formas de aportar:

1. Participa en los directos
Cada espectador suma y ayuda a difundir el mensaje.

2. Comparte el proyecto
Cuantas más personas conozcan la labor de KMAKUS, mayor será el impacto.

3. Colabora con la asociación
Durante el proyecto compartiremos enlaces para:

* Donar
* Apadrinar
* Adoptar
* Ofrecer ayuda como voluntario
* Difundir casos urgentes

4. Contribuye al repositorio
Si eres desarrollador/a, también puedes ayudar proponiendo mejoras, abriendo issues, o enviando PRs.

## Objetivo del Proyecto
El propósito final es doble:

1. Construir una web funcional con Astro, que sirva como ejemplo educativo y recurso gratuito para la comunidad.
2. Apoyar a KMAKUS dándoles más visibilidad y conectando a más personas con su labor.

Queremos demostrar que el código puede tener un sentido más profundo:
crear cosas que generen bienestar, oportunidades y nuevas historias de adopción.

## Tecnologías utilizadas
* Astro
* TypeScript
* SCSS / CSS
* Integraciones y componentes según se avance en los directos
* Herramientas de desarrollo modernas (pnpm, GitHub, Vite…)
